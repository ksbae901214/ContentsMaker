너는 정치 뉴스 분석가다. 아래 어제 발행된 정치 영상 제목과 뉴스 기사 제목들을 같은 이슈끼리 묶어라.

# 입력 형식
- VIDEO: [id={id}] {title}
- NEWS: [id={id}] {title} ({description})

# 출력 규칙
- 같은 사건/인물/정책을 다루는 항목들을 하나의 클러스터로 묶는다.
- 각 클러스터에 대표 토픽 (한글 8~25자)을 지어라. 사건 핵심을 압축.
- 명확히 다른 이슈는 절대 같은 클러스터에 넣지 마라.
- 어느 클러스터에도 속하지 않는 항목은 단일 멤버 클러스터로 만들어라.
- 모든 입력 id는 정확히 하나의 클러스터에 속해야 한다 (누락·중복 금지).

# 출력은 반드시 다음 JSON 형식 한 가지로만 응답하라 (다른 텍스트 금지):
{
  "clusters": [
    {"topic": "대선 양당 후보 부동산 정책 충돌", "member_ids": ["video_1", "video_3", "news_2"]},
    {"topic": "국회의장 사퇴 압박", "member_ids": ["video_2"]}
  ]
}
